📖 如何自建信令服务器

1. 在 Linux 服务器上安装 PeerJS Server:

npm install -g peer

2. 启动服务器:

peerjs --port 9000 --path /myapp

3. 在上方输入服务器地址:

http://你的服务器IP:9000/myapp

4. 如需外网访问，配置端口转发和域名
